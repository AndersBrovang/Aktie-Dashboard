# Interactive Stock EDA Dashboard

A simple and interactive Exploratory Data Analysis (EDA) dashboard built with Python. It allows users to search for any stock ticker and instantly visualize historical trends, moving averages, and risk/volatility.
    
## Key Features
* **Live Data:** Fetches real-time historical market data using Yahoo Finance.
* **Technical Analysis:** Interactive Candlestick charts with 20-day and 50-day Simple Moving Averages (SMA).
* **Volatility Tracking:** Histogram analyzing daily percentage returns to assess risk.
* **Data Export:** Download the cleaned and calculated dataset as a CSV file with a single click.
